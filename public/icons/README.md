## Converting to PNG
To convert SVG icons to PNG format:

1. Use an online SVG to PNG converter
2. Use ImageMagick: `convert icon-192x192.svg icon-192x192.png`
3. Use Inkscape: `inkscape --export-png=icon-192x192.png icon-192x192.svg`
